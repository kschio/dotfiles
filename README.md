## Installation

1. First

```bash
sudo softwareupdate -i -a
xcode-select --install
```

2. Second

```bash
git clone https://github.com/kschio/dotfiles ~/.dotfiles
```

3. Third

```bash
cd ~/.dotfiles
make
```
